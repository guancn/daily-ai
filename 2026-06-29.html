GPT-5.6 Sol 为什么值得关注？

三个层面看这次发布：

  • 模型架构： Sol 引入了 max 深度推理模式+ultra 子Agent加速模式。这意味着 OpenAI 在「更慢更准」和「更快更省」之间给了开发者显式的控制旋钮。
  • 政府审查机制： 美国政府要求逐用户审查 Sol 访问权限，这在 AI 行业是第一次。不是公司自愿的伦理审查，而是政府直接的出口管制式管控。未来前沿模型可能像核材料一样被分级管理。
  • 定价信号： Sol $5/$30 每百万 Token，Terra $2.5/$15（性能对标 GPT-5.5 但便宜一半），Luna $1/$6。这个价位意味着 OpenAI 在同时打高端和性价比两张牌。Sol 在 Cerebras 上可达 750 tokens/s，7月上线。
来源：OpenAI 官方 · Washington Post

Agent 模型路由：workweave/router 为什么值得试？

现在 Agent 面临一个很现实的问题：Claude Code 默认只用 Claude，Codex 只用 GPT，但你明明有更好的模型组合方案。

  • 编程用 Claude Opus，简单查询用 GPT-5.5 Instant，创意写作用 Gemini — 每个模型干自己最擅长的事
  • workweave/router 用 50ms 不到的客户端评分决定本轮路由，不需要额外 API 调用
  • 支持主流 Agent 工具的一键接入：npx @workweave/router --claude 就能让 Claude Code 走路由
  • 自带可观测仪表盘，能看到每轮请求选了哪个模型、为什么

对日调用量大的团队，40-70% 的成本降幅不是理论值。
