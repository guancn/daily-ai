🛡️ MCP Enterprise-Managed Authorization

MCP 协议演进的关键一步。零触摸（Zero-Touch）意味着 Agent 可以自动完成 OAuth 流程，无需开发者手动配置 API Key 或 token。对于想要在企业内部署 Agent 的团队来说，这是重要的安全基础设施。

原文 · Model Context Protocol Blog

🤖 Agent 资源发现：Agent 生态的 DNS 时刻？

Agentic Resource Discovery Specification 解决的是"Agent 怎么知道外面有什么可用"的问题——类似 DNS 之于互联网。与 MCP（定义了 Agent 怎么调用工具）互补，两个协议叠加后，Agent 才能真正实现"发现 → 连接 → 执行"的完整闭环。

原文 · Agentic Resource Discovery

🔬 Token 压缩的真相

Token 压缩听起来很美——减少上下文、降低成本。但作者用实验数据证明：当前方案在精度敏感场景（如代码生成、数学推理）下，压缩带来的 token 节省远不足以弥补准确率损失。对生产环境部署者来说，是一篇及时的泼冷水文章。
