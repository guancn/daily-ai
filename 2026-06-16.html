📖 深度延伸

观点

1. MCP 官方 2026 路线图深度解读

MCP 正从"能用"走向"可靠"。无状态 Streamable HTTP 让传输层可水平扩展、Session Migration 避免服务重启断连、Server Cards 让代理自动发现能力。Working Group 驱动的 SEP 提案机制让社区参与更规范。MCP 正在复制 LSP 的成功路径。

2. 编码 Agent 选型指南：2026 年 6 月版

三句话决策：OpenCode 适合要求模型自由和零月费的开发者；Claude Code 适合需要最可靠 Agent 团队和 1M 上下文的企业；Codex 适合终端优先、需要并行子 Agent 和持久化目标的工作流。三者可以同时用。

3. 2026 年免费 AI 生产力栈推荐

  • 日常对话/写作：DeepSeek V4（免费，中文首选）或 GPT-5.5 Instant
